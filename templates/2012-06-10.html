Number Guessing Game

Create a number guessing game. When the code is executed it should pick a random number between 1 and 100 as the answer. It should also produce a function called guess that accepts a single integer and returns a string.
If the passed argument is lower than the answer it should return "Too low, pick a higher number."
If the passed argument is higher than the answer it should return "Too high, pick a lower number."
If the passed argument is equal to the answer it should return "Hurray! You win!"
